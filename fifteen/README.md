Классическая головоломка «Пятнашки» на Python и Pygame.

## Описание  
- Игровое поле 4×4, одна клетка пустая.  
- Кликайте мышкой на костяшку рядом с пустотой – они поменяются местами.  
- Счётчик ходов в левом верхнем углу.  
- Стартовая раскладка случайная (shuffle).

## Запуск  
```bash
pip install pygame  
python fifteen.py
```

## Управление  
- ЛКМ – передвинуть костяшку.  
- Закрыть окно – выход.

Цель: расставить числа по порядку от 1 до 15, оставив пустую клетку в правом нижнем углу.
