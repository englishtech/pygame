Проекты небольших игр и симуляций с использованием модуля Pygame.  
Каждый проект самодостаточен и может запускаться отдельно.

- **fifteen** - классическая головоломка "Пятнашки" с подсчётом ходов

- **knb** - динамическая симуляция "Камень-Ножницы-Бумага" со 150 спрайтами

- **smiley** - игра-платформер со смайликом, лайками и препятствиями
